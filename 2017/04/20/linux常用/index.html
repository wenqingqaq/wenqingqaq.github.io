vim的一些操作

  1. vim 文件名 编辑文件
  2. gg 光标出现在打开文档的第一行
  3. G 光标出现在最后一行
  4. nG 移动到输入的一行 100G 移动到100行
  5. /word 搜索关键字
  6. ?word 在光标之上查找关键字
  7. n 这个 n 是英文按键。代表重复前一个搜寻的动作。举例来说， 如果刚刚我们执行 /vbird 去向下搜寻 vbird 这个字符串，则按下 n 后，会向下继续搜寻下一个名称为 vbird 的字符串。如果是执行 ?vbird 的话，那么按下 n 则会向上继续搜寻名称为 vbird 的字符串！
  8. N 这个 N 是英文按键。与 n 刚好相反，为『反向』进行前一个搜寻动作。 例如 /vbird 后，按下 N 则表示『向上』搜寻 vbird 。
  9. (数字)x 向后连续删除n个字符
  10. dd 删除游标所在的一行
  11. (数字)dd 删除游标所在下几行
  12. u 撤销上一步的操作 撤销一些错误的修改和删除复制的一些操作
  13. Ctrl+r 恢复上一步被撤销的操作
  14. d1G 删除光标所在到第一行的所有数据
  15. dG 删除光标所在到最后一行的所有数据
  16. d$ 删除游标所在处，到该行的最后一个字符
  17. yy 复制游标所在的那一行(常用)
  18. nyy n 为数字。复制光标所在的向下 n 行，例如 20yy 则是复制 20 行(常用)
  19. y1G 复制游标所在行到第一行的所有数据
  20. yG 复制游标所在行到最后一行的所有数据
  21. y0 复制光标所在的那个字符到该行行首的所有数据
  22. y$ 复制光标所在的那个字符到该行行尾的所有数据
  23. . 不要怀疑！这就是小数点！意思是重复前一个动作的意思。 如果你想要重复删除、重复贴上等等动作，按下小数点『.』就好了！ (常用)
